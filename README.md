# Ziki's Simple Fractions Class


#### This is a fractions class in C++

## Installation
Clone the repository
```sh
git clone https://github.com/Zikithezikit/CppFractions
```
Get into the folder
```sh
cd ./CppFractions
```
Run a compiler 
```sh
g++ -o app ./main.cpp ./FractionsClass.h ./fractionsClass.cpp
```
Run the executable 
```sh
./app.exe
```
